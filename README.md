# postal-code-macro

Claude Code に Excel VBA マクロを修正してもらうと実際どうなるのか──その **before / after を実物のブックで公開**しているリポジトリです。

Qiita 連載「Excel VBA × Claude Code」の記事のサンプルです。

- `修正前県内郵便.xlsm` ── **before**（修正前のまま）
- `県内郵便.xlsm` ── **after**（Claude Code + [shu-vba-manager](https://github.com/shu1551/shu-vba-manager) で修正した後）

## どんなブックか

秋田県内の郵便番号検索ブックです。シート「県内郵便」のセル **B1 に地名を入力**すると、マクロが自動で動き、3行目に住所・郵便番号・読みを表示します。

```
B1 に「秋田市山王」と入力
　↓
B3: 秋田県秋田市山王　C3: 010-0951　D3: あきたけんあきたしさんのう
```

1件に絞り込めないときはオートフィルタで候補を表示します。データは日本郵便が公開している秋田県内の郵便番号データ（約2,300件）です。

## before / after の主な違い

- 変数名：`ZZZ` / `AAA` / `BBB` … → `検索語` / `照合パターン` / `データ範囲` …（日本語変数名＋意味コメント）
- 検索ロジック：手動メンテナンスの例外名簿103件（優先リストシート）→ 末尾の番地類を剥がして親地区を判定するロジックに置き換え、名簿ごと廃止
- `SendKeys "{DOWN}"` の全廃、`Selection` 依存の解消、フィルタ解除の確実化、クリップボード書き込みの再試行、`Option Explicit` ほか

詳しくは記事をどうぞ（公開後にリンクを追記）。

## after 限定：マクロ一覧メニュー

修正後のブックでは **Ctrl+Shift+M** でマクロ一覧フォームが開きます。

- テキストボックスに入力すると一覧が絞り込まれます
- 選んで Enter / ダブルクリックで実行
- **右クリックで VBE のそのマクロのコードへジャンプ**（AI が直したコードを直接見て回れます）

※ マクロ一覧は VBA プロジェクトをマクロから読む作りのため、利用には Excel のトラストセンターで「VBA プロジェクト オブジェクト モデルへのアクセスを信頼する」を有効にする必要があります。設定しない場合も Alt+F11 で VBE を開けばコードは見られます。

## 注意

- どちらのブックもマクロ入り (.xlsm) です。開くときはマクロを有効にしてください
- ダウンロードしたファイルは Windows にブロックされることがあります（右クリック → プロパティ → 「許可する」）
